## Codebase Patterns
- **Entity package:** com.student.entity - @Data Lombok, @TableName, @TableId(type=IdType.AUTO)
- **Mapper package:** com.student.mapper - extends BaseMapper<T>
- **Service package:** com.student.service (interface) + com.student.service.impl (implementation)
- **Controller package:** com.student.controller - @RestController, uses Result<T> wrapper, @RequestBody Map<String,String>
- **Common package:** com.student.common - Result<T> unified response
- **Util package:** com.student.util - JwtUtil
- **Filter package:** com.student.filter - JwtFilter (OncePerRequestFilter)
- **Config:** SecurityConfig, CorsConfig, MybatisPlusConfig (pagination)
- **CascadeService:** @Transactional, uses LambdaQueryWrapper for conditional deletes
- **Test:** H2 in-memory DB (MODE=MySQL), schema-h2.sql, spring.sql.init.schema-locations, @ActiveProfiles("test")
- **Test cleanup:** @BeforeEach manually clears all tables (grade→selection→student/teacher)
- **Naming:** Entity fields use camelCase (maps to snake_case via mybatis-plus config)
- **Auth errors:** All auth failures throw RuntimeException with code string, controller maps to generic 401
- **Controller pattern:** Use @RequestBody Map<String,String> for flexibility, PasswordEncoder for passwords
