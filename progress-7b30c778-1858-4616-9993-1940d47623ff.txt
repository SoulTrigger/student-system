## Codebase Patterns
- **Entity package:** com.student.entity - @Data Lombok, @TableName, @TableId(type=IdType.AUTO)
- **Mapper package:** com.student.mapper - extends BaseMapper<T>
- **Service package:** com.student.service (interface) + com.student.service.impl (implementation)
- **Controller package:** com.student.controller - @RestController, uses Result<T> wrapper
- **Common package:** com.student.common - Result<T> unified response
- **CascadeService:** @Transactional, uses LambdaQueryWrapper for conditional deletes
- **Test:** H2 in-memory DB (MODE=MySQL), schema-h2.sql, @ActiveProfiles("test")
- **Test cleanup:** @BeforeEach manually clears all tables (grade→selection→student/teacher/course)
- **Controller CRUD pattern:** list(pagination+模糊+filters), add(auto-id+validation), update(optional fields+validation), delete(removeById+cascade)
- **Course controller:** Uses @RequestBody Map<String,Object> for numeric credit parsing via ((Number) body.get("credit")).intValue()
